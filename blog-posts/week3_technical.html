The Popularity of Javascript

There are threee main reasons why Javascript is so popular.

  • Javascript is easy to learn
  • Javascript is powerful
  • Javascript is ubiqitous

Javascript is easy to learn

It is fairly easy to learn and get working on a website quickly. This lets users develop programs and ideas and implement them quickly and see the results live. Javascript is also easy to edit and debug.

Javascript is powerful

It is simple to add to your programs and loads onto the browser so only requires a functioning browser to work. Like some other languages it has an object oriented structure. It is used mostly on the client side freeing up the server for other tasks. People are using it for more advanced tasks and pages are becoming more Javascript dependent.

Javascript is asynchronous. Asynchronous means that Javascript, unlike other languages, does NOT need to process information sequentially. It can work on various tasks simultaneously, it does NOT need to finish one task before moving onto the next.

Javascript is easily extensible and employs some very useful libraries like jQuery, Prototype and Dojo. These libraries allow users to easily manipulate pages with just a few lines of code.

Javascript is ubiqitous

Most browsers have some form of Javascript. The exact figures are debatable but approximately 95% of your user's browsers will support Javascript in some form.

Until around 2009 Javascript was mostly used for the DOM. Now Javascript is being used to write physics simulations, 2d graphics, 3d graphics, audio processing and synthesizing, real-time video processing, codec decoding, etc. So it is increasingly being used for almost everything!
