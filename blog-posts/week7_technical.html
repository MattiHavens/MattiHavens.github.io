SQL Injection





SQL Injection is a way for malicious hackers to get into a SQL database and alter and/or steal information. After seeing how it is done and how commonly it has occurred I am surprised it can still happen. As a matter of fact as stated by Veracode"With over 20% of all web vulnerabilities being attributed to SQL Injection, this is the 2nd most common software vulnerability."
